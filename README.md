# Mumbai-CO2-Prediction

An end-to-end Big Data project for predicting CO2 levels in specific areas of Mumbai using the ARIMA (Auto-Regressive Integrated Moving Average) time-series model. This project demonstrates a full-stack AI pipeline: distributed data processing (Spark simulation), a portable structured database (SQLite), and a professional-grade dashboard.🏗️ Project ArchitectureData Processing: PySpark for handling and cleaning large-scale sensor data.Modeling: ARIMA via statsmodels for time-series forecasting.Storage: SQLite for a 100% offline, zero-configuration relational database.Backend: FastAPI (Python) for high-speed data serving.Frontend: React.js with Tailwind CSS (Deep Navy & Electric Blue Theme).🚀 Step-by-Step Execution GuideTo run this project, you will need three separate terminal windows open in the Mumbai-CO2-Prediction root directory.🟢 Terminal 1: Setup & Data PipelinePurpose: Initialize the environment and populate the database.

Create & Activate Virtual Environment:
Bash
python -m venv venv

# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

Install All Dependencies:
Bashpip 
install -r requirements.txt

Run Big Data Cleaning (Spark):
Bash
python big_data_processing/spark_cleaner.py
Run AI Modeling (ARIMA):
Bash
python notebooks/arima_research.py
Result: This generates the backend/mumbai_co2.db file with both historical and predicted data.🔵 Terminal 2: Backend APIPurpose: Serve data from the database to the web interface.Activate Virtual Environment:Windows: venv\Scripts\activateMac/Linux: source venv/bin/activateLaunch the API:Bashcd backend
uvicorn main:app --reload
The API will be live at: http://127.0.0.1:8000🟡 Terminal 3: Frontend DashboardPurpose: Visualize the predictions on a professional UI.Navigate and Install Node modules:Bashcd frontend
npm install
Launch the Dashboard:Bashnpm run dev
Open the URL provided (usually http://localhost:5173) in your browser.📊 How to Use the DashboardArea Selection: Use the sidebar to switch between specific Mumbai areas (e.g., Bandra, Kurla, Colaba).Historical View: Analyze the past CO2 trends processed via Spark.Forecast View: View the 7-day predicted $CO_2$ levels generated by the ARIMA model.📁 File StructurePlaintextMumbai-CO2-Prediction/

🛠️ Technical Implementation DetailsStationarity: The model performs an Augmented Dickey-Fuller (ADF) test to determine the differencing parameter ($d$).Big Data Logic: Spark is used to handle outlier detection and null-value imputation using mean-averaging across the dataset.UI Palette: Professionally designed using Deep Navy (#0a192f) and Electric Blue (#00d4ff) for a minimalist technical aesthetic.Note to Reviewer: This project is designed to run entirely offline. All data is contained within the included SQLite database file. Ensure Node.js (v18+) and Python (v3.9+) are installed.
